# TypeScript Fullstack Developer — DIO

Repositório de estudos da formação **TypeScript Fullstack Developer da DIO**, com foco em desenvolvimento fullstack utilizando **TypeScript**, **React**, **Node.js**, **Express**, **APIs REST**, **Orientação a Objetos** e projetos práticos para portfólio.

Este repositório foi criado para documentar minha jornada de aprendizado, organizar exercícios e desafios, e construir progressivamente projetos que possam fortalecer meu GitHub e apoiar minha transição profissional para a área de tecnologia.

---

## Sobre a Formação

A formação **TypeScript Fullstack Developer** tem como objetivo desenvolver habilidades práticas com TypeScript no Front-End e no Back-End.

A trilha aborda:

- fundamentos do TypeScript;
- Programação Orientada a Objetos;
- React com TypeScript;
- componentes tipados;
- propriedades tipadas;
- manipulação de estado;
- consumo de APIs;
- Node.js com TypeScript;
- Express;
- desenvolvimento de APIs REST;
- boas práticas de Back-End;
- persistência com TypeORM;
- autenticação com JWT;
- desafios de código;
- desafios de projeto.

Segundo a página da formação na DIO, a trilha possui aproximadamente **44 horas**, **18 cursos**, **4 desafios de projeto** e **5 desafios de código**.

## Por que TypeScript?

TypeScript melhora o desenvolvimento com JavaScript ao adicionar tipagem estática e melhor suporte de ferramentas.

Alguns benefícios:

- código mais seguro;
- melhor autocomplete;
- refatoração mais confiável;
- contratos mais claros em funções;
- maior escalabilidade;
- redução de erros em tempo de execução;
- melhor colaboração em projetos maiores.
